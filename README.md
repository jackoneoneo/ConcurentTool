> 9. 线程的优势：
> - 线程可以有效的降低程序的开发成本和维护成本，同时提升复杂应用程序的性能。
> - 线程能将大部分的异步工作流转换成串行工作流，因此能更好的模拟人类的工作方式和交互方式。
